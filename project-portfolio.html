Portfolio Platform

Full-Stack Project Management & Portfolio App

Portfolio Platform is a full-stack web application that allows users to create an account, manage projects, edit profile information, and share a public portfolio page.

GitHub Repo: View on GitHub →

Project Overview

The app separates private dashboard tools from the public portfolio view. Users can log in, add or edit projects, update their profile, and generate a shareable portfolio link.

Core Features
